Quick Start

几分钟内开始使用 Orchestra

1

Clone Repository

git clone https://github.com/wayyoungboy/Orchestra.git
2

Start Backend

cd backend && make build && make run
3

Start Frontend

cd frontend && npm install && npm run dev

Requirements

  • Go 1.21+
  • Node.js 18+
  • SQLite3
